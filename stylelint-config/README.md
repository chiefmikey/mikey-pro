## **@mikey-pro/stylelint-config**

A preset Stylelint configuration

## Usage

### Install

```shell
npm i -D mikey-pro
```

### Configure

Extend to Stylelint in `package.json`:

```json
{
  "stylelint": {
    "extends": ["@mikey-pro/stylelint-config"]
  }
}
```
