# @mikey-pro/eslint-config-svelte

Svelte-specific ESLint configuration extending the core Mikey Pro style guide.

## Installation

```bash
npm install @mikey-pro/eslint-config-svelte
```

## Usage

```javascript
// eslint.config.js
export { default } from '@mikey-pro/eslint-config-svelte';
```

## Features

- Svelte component structure validation
- Reactive statement rules
- Store usage best practices
- Svelte 5 compatibility
- Extends core Mikey Pro configuration

See [Mikey Pro](https://github.com/chiefmikey/mikey-pro) for full documentation.
