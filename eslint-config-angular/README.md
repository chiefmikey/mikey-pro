## **@mikey-pro/eslint-config-angular**

A preset ESLint Angular configuration

## Usage

### Install

```shell
npm i -D mikey-pro @mikey-pro/eslint-config-angular
```

### Configure

Extend to ESLint in `package.json`:

```json
{
  "eslintConfig": {
    "extends": ["@mikey-pro/eslint-config-angular"]
  }
}
```
