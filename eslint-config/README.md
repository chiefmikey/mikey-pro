## **@mikey-pro/eslint-config**

_A preset ESLint configuration_

## Usage

### Install

```shell
npm i -D mikey-pro
```

### Configure

Extend to ESLint in `package.json`:

```json
{
  "eslintConfig": {
    "extends": ["@mikey-pro/eslint-config"]
  }
}
```
