## **@mikey-pro/prettier-config**

_A preset Prettier configuration_

## Usage

### Install

```shell
npm i -D @mikey-pro/prettier-config

yarn add -D @mikey-pro/prettier-config
```

### Configure

Extend to Prettier in `package.json`:

```json
{
  "prettier": {
    "extends": [
      "@mikey-pro/prettier-config"
    ]
  }
}
```
